Prueba de la ruta POST /registro 

Para probar la API use bruno

Configuracion de la peticion: 
- Metodo: POST
- URL: http://localhost:3000/registro 
- Encabezado: Content-Type: application/json 
- Body (JSON) 

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Respuesta recibida del servidor: 
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor recibio los datos correctamente gracias al middleware express.json(), que permite leer el cuerpo de la 
peticion POST. Los datos se accedieron mediante req.body.nombre y req.body.mensaje y 
fueron devueltos al cliente en formato JSON como confirmacion.

Esta prueba confirma que la API puede recibir datos en formato JSON y responder correctamente, 
lo que valida el funcionamiento del middleware y la ruta POST.
